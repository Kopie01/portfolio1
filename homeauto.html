Home Automation Interface

This is an user interface design for a New Zealand home automation system called SafeWhare (Whare being the Maori word for house). For this project I focused on a simple but friendly design, using a warm blue to show security but also give it a friendly, homely feel. The functional design is for a mobile, but I have also designed a layout option for a tablet or smart TV as shown below. Click on the SafeWhare logo to view the mobile site.
